–ò–Ω—Ç–µ–≥—Ä–∏—Ä–æ–≤–∞–Ω–∏–µ ‚Äî –æ–¥–Ω–∞ –∏–∑ –∫–ª—é—á–µ–≤—ã—Ö –æ–ø–µ—Ä–∞—Ü–∏–π –≤ –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–æ–º –∞–Ω–∞–ª–∏–∑–µ. –ï—Å–ª–∏ –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç, *–∫–∞–∫ –±—ã—Å—Ç—Ä–æ* —Ñ—É–Ω–∫—Ü–∏—è –∏–∑–º–µ–Ω—è–µ—Ç—Å—è, —Ç–æ –∏–Ω—Ç–µ–≥—Ä–∞–ª –ø–æ–∑–≤–æ–ª—è–µ—Ç *–Ω–∞–∫–∞–ø–ª–∏–≤–∞—Ç—å* —ç—Ç–∏ –∏–∑–º–µ–Ω–µ–Ω–∏—è. –ß–∞—â–µ –≤—Å–µ–≥–æ –∏–Ω—Ç–µ–≥—Ä–∏—Ä–æ–≤–∞–Ω–∏–µ —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—é—Ç –∫–∞–∫ –æ–ø–µ—Ä–∞—Ü–∏—é, –æ–±—Ä–∞—Ç–Ω—É—é –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–æ–≤–∞–Ω–∏—é.

–ù–µ–æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–π –∏–Ω—Ç–µ–≥—Ä–∞–ª

–ù–µ–æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–π –∏–Ω—Ç–µ–≥—Ä–∞–ª ‚Äî —ç—Ç–æ –ø–æ —Å—É—Ç–∏ "–∞–Ω—Ç–∏-–ø—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è". –ï—Å–ª–∏ —É –Ω–∞—Å –µ—Å—Ç—å —Ñ—É–Ω–∫—Ü–∏—è $f(x)$, —Ç–æ –µ–µ –Ω–µ–æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–º –∏–Ω—Ç–µ–≥—Ä–∞–ª–æ–º –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è —Ç–∞–∫–∞—è —Ñ—É–Ω–∫—Ü–∏—è $F(x)$, –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è –∫–æ—Ç–æ—Ä–æ–π —Ä–∞–≤–Ω–∞ $f(x)$.

–¢–∞–∫–∞—è —Ñ—É–Ω–∫—Ü–∏—è $F(x)$ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –ø–µ—Ä–≤–æ–æ–±—Ä–∞–∑–Ω–æ–π –¥–ª—è $f(x)$.

–ù–∞–ø—Ä–∏–º–µ—Ä, –µ—Å–ª–∏ $f(x) = 2x$, —Ç–æ –µ–µ –ø–µ—Ä–≤–æ–æ–±—Ä–∞–∑–Ω–æ–π –±—É–¥–µ—Ç $F(x) = x^2$, –ø–æ—Ç–æ–º—É —á—Ç–æ –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è $(x^2)' = 2x$.

–í–∞–∂–Ω—ã–π –º–æ–º–µ–Ω—Ç: –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è –æ—Ç $x^2 + 5$ —Ç–∞–∫–∂–µ —Ä–∞–≤–Ω–∞ $2x$. –ò –æ—Ç $x^2 - 100$ —Ç–æ–∂–µ. –ü—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è –ª—é–±–æ–π –∫–æ–Ω—Å—Ç–∞–Ω—Ç—ã (—á–∏—Å–ª–∞) —Ä–∞–≤–Ω–∞ –Ω—É–ª—é. –ü–æ—ç—Ç–æ–º—É, –∫–æ–≥–¥–∞ –º—ã –Ω–∞—Ö–æ–¥–∏–º –Ω–µ–æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–π –∏–Ω—Ç–µ–≥—Ä–∞–ª, –º—ã –≤—Å–µ–≥–¥–∞ –¥–æ–±–∞–≤–ª—è–µ–º "–∫–æ–Ω—Å—Ç–∞–Ω—Ç—É –∏–Ω—Ç–µ–≥—Ä–∏—Ä–æ–≤–∞–Ω–∏—è" $C$.

$$ \int f(x) \,dx = F(x) + C $$

–ó–¥–µ—Å—å $ \int $ ‚Äî –∑–Ω–∞–∫ –∏–Ω—Ç–µ–≥—Ä–∞–ª–∞, $f(x)$ ‚Äî –ø–æ–¥—ã–Ω—Ç–µ–≥—Ä–∞–ª—å–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è, $dx$ ‚Äî —É–∫–∞–∑—ã–≤–∞–µ—Ç –Ω–∞ —Ç–æ, –ø–æ –∫–∞–∫–æ–π –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –∏–¥–µ—Ç –∏–Ω—Ç–µ–≥—Ä–∏—Ä–æ–≤–∞–Ω–∏–µ.

–¢–∞–±–ª–∏—Ü–∞ –æ—Å–Ω–æ–≤–Ω—ã—Ö –∏–Ω—Ç–µ–≥—Ä–∞–ª–æ–≤

  • $ \int 0 \,dx = C $
  • $ \int x^n \,dx = \frac{x^{n+1}}{n+1} + C $ (–¥–ª—è $n \neq -1$)
  • $ \int \frac{1}{x} \,dx = \ln|x| + C $
  • $ \int e^x \,dx = e^x + C $
  • $ \int a^x \,dx = \frac{a^x}{\ln a} + C $
  • $ \int \sin(x) \,dx = -\cos(x) + C $
  • $ \int \cos(x) \,dx = \sin(x) + C $

–û–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–π –∏–Ω—Ç–µ–≥—Ä–∞–ª

–û–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–π –∏–Ω—Ç–µ–≥—Ä–∞–ª –∏–º–µ–µ—Ç —á–µ—Ç–∫–∏–µ –≥—Ä–∞–Ω–∏—Ü—ã (–ø—Ä–µ–¥–µ–ª—ã) –∏–Ω—Ç–µ–≥—Ä–∏—Ä–æ–≤–∞–Ω–∏—è. –ì–µ–æ–º–µ—Ç—Ä–∏—á–µ—Å–∫–∏ –æ–Ω –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π –ø–ª–æ—â–∞–¥—å –ø–æ–¥ –≥—Ä–∞—Ñ–∏–∫–æ–º —Ñ—É–Ω–∫—Ü–∏–∏ $f(x)$ –Ω–∞ –æ—Ç—Ä–µ–∑–∫–µ –æ—Ç $a$ –¥–æ $b$.

–î–ª—è –µ–≥–æ –≤—ã—á–∏—Å–ª–µ–Ω–∏—è –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è —Ñ–æ—Ä–º—É–ª–∞ –ù—å—é—Ç–æ–Ω–∞-–õ–µ–π–±–Ω–∏—Ü–∞, –∫–æ—Ç–æ—Ä–∞—è —Å–≤—è–∑—ã–≤–∞–µ—Ç –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–π –∏–Ω—Ç–µ–≥—Ä–∞–ª —Å –ø–µ—Ä–≤–æ–æ–±—Ä–∞–∑–Ω–æ–π:

$$ \int_a^b f(x) \,dx = F(b) - F(a) $$

–ó–¥–µ—Å—å $a$ ‚Äî –Ω–∏–∂–Ω–∏–π –ø—Ä–µ–¥–µ–ª, $b$ ‚Äî –≤–µ—Ä—Ö–Ω–∏–π –ø—Ä–µ–¥–µ–ª, –∞ $F(x)$ ‚Äî –ª—é–±–∞—è –ø–µ—Ä–≤–æ–æ–±—Ä–∞–∑–Ω–∞—è –¥–ª—è $f(x)$. –ö–æ–Ω—Å—Ç–∞–Ω—Ç–∞ $C$ –∑–¥–µ—Å—å –Ω–µ –Ω—É–∂–Ω–∞, —Ç–∞–∫ –∫–∞–∫ –æ–Ω–∞ —Å–æ–∫—Ä–∞—Ç–∏—Ç—Å—è –ø—Ä–∏ –≤—ã—á–∏—Ç–∞–Ω–∏–∏: $(F(b) + C) - (F(a) + C) = F(b) - F(a)$.

–ú–µ—Ç–æ–¥—ã –∏–Ω—Ç–µ–≥—Ä–∏—Ä–æ–≤–∞–Ω–∏—è

–ù–∞—Ö–æ–∂–¥–µ–Ω–∏–µ –∏–Ω—Ç–µ–≥—Ä–∞–ª–æ–≤ (–æ—Å–æ–±–µ–Ω–Ω–æ –Ω–µ–æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã—Ö) ‚Äî –∑–∞–¥–∞—á–∞ –ø–æ—Ä–æ–π –Ω–µ—Ç—Ä–∏–≤–∏–∞–ª—å–Ω–∞—è. –°—É—â–µ—Å—Ç–≤—É–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ –º–µ—Ç–æ–¥–æ–≤, –Ω–æ –¥–≤–∞ —Å–∞–º—ã—Ö —Ñ—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω—ã—Ö:

1. –ú–µ—Ç–æ–¥ –∑–∞–º–µ–Ω—ã (–ü–æ–¥—Å—Ç–∞–Ω–æ–≤–∫–∞)

–≠—Ç–æ—Ç –º–µ—Ç–æ–¥ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è, –∫–æ–≥–¥–∞ –ø–æ–¥—ã–Ω—Ç–µ–≥—Ä–∞–ª—å–Ω–æ–µ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ –º–æ–∂–Ω–æ —É–ø—Ä–æ—Å—Ç–∏—Ç—å, –≤–≤–µ–¥—è –Ω–æ–≤—É—é –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é. –û–Ω –æ—Å–Ω–æ–≤–∞–Ω –Ω–∞ –ø—Ä–∞–≤–∏–ª–µ –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–æ–≤–∞–Ω–∏—è —Å–ª–æ–∂–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏.

–ü—Ä–∏–º–µ—Ä: –ù–∞–π–¥–µ–º $ \int (2x + 3)^5 \,dx $.

–ü—É—Å—Ç—å $t = 2x + 3$. –¢–æ–≥–¥–∞ $dt = (2x + 3)' \,dx = 2 \,dx$, –æ—Ç–∫—É–¥–∞ $dx = \frac{dt}{2}$.

–ü–æ–¥—Å—Ç–∞–≤–ª—è–µ–º: $ \int t^5 \frac{dt}{2} = \frac{1}{2} \int t^5 \,dt = \frac{1}{2} \cdot \frac{t^6}{6} + C = \frac{t^6}{12} + C $.

–í–æ–∑–≤—Ä–∞—â–∞–µ–º—Å—è –∫ $x$: $ \frac{(2x + 3)^6}{12} + C $.

2. –ò–Ω—Ç–µ–≥—Ä–∏—Ä–æ–≤–∞–Ω–∏–µ –ø–æ —á–∞—Å—Ç—è–º

–≠—Ç–æ—Ç –º–µ—Ç–æ–¥ –æ—Å–Ω–æ–≤–∞–Ω –Ω–∞ –ø—Ä–∞–≤–∏–ª–µ –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–æ–≤–∞–Ω–∏—è –ø—Ä–æ–∏–∑–≤–µ–¥–µ–Ω–∏—è. –û–Ω –ø—Ä–∏–º–µ–Ω—è–µ—Ç—Å—è, –∫–æ–≥–¥–∞ –ø–æ–¥ –∏–Ω—Ç–µ–≥—Ä–∞–ª–æ–º —Å—Ç–æ–∏—Ç –ø—Ä–æ–∏–∑–≤–µ–¥–µ–Ω–∏–µ –¥–≤—É—Ö —Ñ—É–Ω–∫—Ü–∏–π.

–§–æ—Ä–º—É–ª–∞:

$$ \int u \,dv = uv - \int v \,du $$

–ü—Ä–∏–º–µ—Ä: –ù–∞–π–¥–µ–º $ \int x \cdot \cos(x) \,dx $.

–ü—É—Å—Ç—å $u = x$ (—Ç–æ, —á—Ç–æ –º—ã –±—É–¥–µ–º –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–æ–≤–∞—Ç—å) –∏ $dv = \cos(x) \,dx$ (—Ç–æ, —á—Ç–æ –±—É–¥–µ–º –∏–Ω—Ç–µ–≥—Ä–∏—Ä–æ–≤–∞—Ç—å).

–¢–æ–≥–¥–∞ $du = dx$ –∏ $v = \int \cos(x) \,dx = \sin(x)$.

–ü–æ–¥—Å—Ç–∞–≤–ª—è–µ–º –≤ —Ñ–æ—Ä–º—É–ª—É:

$ \int x \cdot \cos(x) \,dx = x \cdot \sin(x) - \int \sin(x) \,dx = x \cdot \sin(x) - (-\cos(x)) + C = x \cdot \sin(x) + \cos(x) + C $
